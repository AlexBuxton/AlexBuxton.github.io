Definition

To define a metric space, we first need to define a metric. Simply put, a metric is a function d : X² → ℝ that measures the distance between two points in a set. The most obvious example is the function d : ℝ² → ℝ given by d(x, y) = |x - y|, which is the standard metric used to measure distance on ℝ. To make this notion more formal, we define a metric as a function d : X² → ℝ, satisfying the three properties:

  • Positivity: d(x, y) ≥ 0, with d(x, y) = 0 if and only if x = y
  • Symmetry: d(x, y) = d(y, x)
  • Triangle inequality: d(x, y) + d(y, z) ≥ d(x, z)

A metric space is then simply a set X endowed with some metric d. Formally we denote this metric space as (X, d), but when it is obvious which metric we are using, we often just denote it by X.

Fun Facts

These are my three favourite facts about metric spaces:

  1. The inequality
    |d(x, y) - d(x, z)| ≤ d(y, z)
    is known as the reverse triangle inequality. It is equivalent to the triangle inequality.

  2. When presented with a problem about metric spaces, it can be tempting to use geometrical intuition. For many false theorems which appear true based on geometrical intuition, you can use the discrete metric as a counter example. The discrete metric d : X² → ℝ is given by
    d(x, y) = 1 if x ≠ y, d(x, y) = 0 if x = y
    where X is an arbitrary set.

  3. Given a norm ‖·‖, d(x, y) := ‖x - y‖ defines a metric. If you aren't familiar with norms, a norm is defined as a map ‖·‖ : V → [0, ∘), where V is a vector space, satisfying:

    • ‖x‖ ≥ 0, with ‖x‖ = 0 if and only if x = 0
    • ∀ λ ∈ ℝ, x ∈ V, ‖λx‖ = |λ|‖x‖
    • ‖x‖ + ‖y‖ ≥ ‖x + y‖

Theorem

The reverse triangle inequality
|d(x, y) - d(x, z)| ≤ d(y, z)
is equivalent to the triangle inequality
d(x, y) + d(y, z) ≥ d(x, z).

Proof

By the triangle inequality,
d(x, z) + d(y, z) ≥ d(x, y)
and
d(x, y) + d(y, z) ≥ d(x, z).

So
|d(x, y) - d(x, z)| ≤ d(y, z).
