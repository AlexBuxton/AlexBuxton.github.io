Definition

To define a metric space, we first need to define a metric. Simply put, a metric is a function d : X² → ℝ that measures the distance between two points in a set. The most obvious example is the function d : ℝ² → ℝ given by d(x, y) = |x - y|, which is the standard metric used to measure distance on ℝ. To make this notion more formal, we define a metric as a function d : X² → ℝ, satisfying the three properties:

  1. Positivity: ∀ x, y ∈ X, d(x, y) ≥ 0, with d(x, y) = 0 if and only if x = y
  2. Symmetry: ∀ x, y ∈ X, d(x, y) = d(y, x)
  3. Triangle inequality: ∀ x, y, z ∈ X, d(x, y) + d(y, z) ≥ d(x, z)

A metric space is then simply a set X endowed with some metric d. Formally we denote this metric space as (X, d), but when it is obvious which metric we are using, we often just denote it by X.
